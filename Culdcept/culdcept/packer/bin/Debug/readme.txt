                              FaTILEty - Versão 1.2c
                                The Tiles Manager


------=====Introdução=====------

    Fatilety é um editor de TILES que suporta múltiplos formatos
    gráficos de diversos sistemas existentes, algumas de suas
    funções são baseadas em funções dos Tile editors mais utilizados
    como o Tile Layer e o TilED, porém ele tem algumas funções inovadoras
    que facilitaram a edição dos gráficos principalmente para grandes
    modificações.


------======Funções=======------

    * Multi Idioma
    * Sistema de Favoritos
    * Corretor de Checksum de GB/GBC/GBA
    * Conversor SMD/BIN BIN/SMD com backup de Header
    * Importação/Exportação de BMP
    * Importação/Exportação de Tabela de Endereços (Clipboard)
    * Paleta Customizável de 2 a 256 cores
    * Gerador de Paleta Aleatória
    * Importação/Exportação de Paletas (Formato nativo do FaTILEty ".fpl")
    * Importação de Paletas de Bitmaps
    * Edição Múltipla de Tiles (4x4 tiles no máximo)
    * Editor e Tela de Tiles Redimensináveis
    * 2 modos de edição, Cópia (Apenas faz uma cópia do tile editado) e Ponteiro
      (Cria um ponteiro para o endereço da rom onde se encontra o tile editado,
      atualizando-o automaticamente)
    * Tela de Tiles com Zoom de 1x até 6x
    * Algumas outras coisas que não lembro agora...

  *Ferramentas de Desenho:
    + Desenho a Mão Livre (Lápis)
    + Linha
    + Retângulo
    + Ellipse
    + Selecionador de Cor
    + Preenchimento

  *Formatos Suportados:
 
    Abaixo uma lista com os formatos suportados pelo FaTILEty e exemplos de sistemas
    que os usam, porém, oficialmente ou não oficialmente outros sistemas também podem
    usá-los.
   
    + 1bpp - Usado por vários sistemas como Atari 2600, Intelivision e ColecoVision, 
      usado também em outros sistemas para economizar espaço
    + 2bpp NES - Usado somente no NES
    + 2bpp Virtual Boy - Usado somente no Virtual Boy
    + 2bpp Neo Geo Pocket - Usado somente no Neo Geo Pocket
    + 2bpp GB, GBC - Usado no Game Boy e no Game Boy Color, algumas roms do Wonder Swan
      também utilizam este formato que também é usado em alguns jogos do SNES
    + 3bpp SNES - Usado no Zelda 3, é uma espécie de compactação, certamente deve ser 
      utilizado por outros jogos.
    + 4bpp SMS, GG, WSC - Usado no Sega Master System, no Sega Game Gear e em algumas 
      roms do Wonder Swan Color.
    + 4bpp Genesis - Usado somente no Sega Genesis
    + 4bpp SNES - Usado no somente no SNES
    + 4bpp GBA - Usado no Game Boy Advance
    + 8bpp SNES - Usado no somente no SNES
    + 8bpp SNES Mode 7, GBA - Usado no SNES e no Game Boy Advance
